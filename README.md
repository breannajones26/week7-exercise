# Dance Class Tracker

A simple web app to track dance classes you've taken.

## Features

- Log classes with date, style, instructor, and notes
- View class history sorted by date
- See all your instructors and how many classes with each
- Browse dance styles
- Data saved locally in your browser

## Getting Started

1. Open `index.html` in any web browser
2. Click "+ Add Class" to log your first class
3. That's it!

## Files

- `index.html` - Main page structure
- `styles.css` - All styling
- `script.js` - App logic and localStorage
- `PLAN.md` - Feature plan and data model

## Clearing Data

To reset all data, open browser DevTools (F12) → Application tab → localStorage → Clear.
